Tools

These are the tools and technologies I use on my computers, for software development and more.

Editors

Neovim

My primary editor of choice is Neovim. I use many customizations to fit my workflow. My dotfiles can be found at this link.

Lately, I've been trying out LazyVim, which is a Neovim distribution. I find it very easy to work with, although I don't understand half of the included plugins, or how to customize them.

Cursor

I occasionally use the Cursor editor. It is a VS Code fork with built in AI features to generate code. I don't like the VS Code experience in general. But it has its uses. I use Cursor with a Vim emulation plugin.

I like the code generation features. But it doesn't look like I'll move permanently away from Neovim in the foreseeable future.

Terminal

I do most of my coding in the termial. The terminal emulator I use is WezTerm, on top of which I run tmux. I have configured my tmux to help my coding and file navigation habits.
